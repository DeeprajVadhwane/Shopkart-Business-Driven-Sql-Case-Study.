# ShopKart Business-Driven SQL Case Studies

Retail sales performance analysis using SQL with real-world business scenarios. This repository contains two complete case studies designed for interview preparation and practical learning.

---

## 📚 Case Studies

### 📘 Case Study 1: Core SQL Analytics (Up to CTE)

This case study covers all essential SQL concepts from beginner to advanced, including:

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- JOINs
- CASE WHEN
- Subqueries
- Common Table Expressions (CTEs)

### 📙 Case Study 2: Advanced SQL Window Functions

This case study focuses entirely on window functions and advanced analytical SQL.

Topics covered:

- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- NTILE()
- LAG()
- LEAD()
- FIRST_VALUE()
- LAST_VALUE()
- Running Totals
- Moving Averages
- Percentage Contribution

## 🎯 Learning Outcomes

By completing both case studies, you will learn how to:

- Solve real-world business problems using SQL
- Analyze retail sales and customer behavior
- Write optimized SQL queries
- Master advanced window functions
- Prepare for SQL interviews

---

## 🧠 Recommended Learning Path

1. Complete Case Study 1 to build a strong SQL foundation.
2. Move to Case Study 2 to master window functions.
3. Practice all questions and review the solutions.
4. Build dashboards in Power BI or Tableau.
